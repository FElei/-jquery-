# -jquery-
在某些场景一个div里面需要装入解释信息，比如：某些商城里面，在一个盒子里需要备注商品的信息和一些解释，这些基本都是被隐藏了的，需要鼠标进入才会出现，该插件是基于jQuery的鼠标跟随插件，顾名思义，鼠标从哪里进入显示信息的盒子也从哪里进入显示区，鼠标离开，该盒子也从相应的方向离开。
